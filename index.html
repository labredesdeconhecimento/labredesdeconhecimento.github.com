Python para Administradores de Redes Linux

Voltado para administradores de sistemas e demais entusiastas da área de Redes de Computadores, o curso abordará o emprego prático e teórico da linguagem de programação Python na resolução de problemas e desafios da administração de redes.

Programação para Dispositivos Móveis com Android

Voltado para desenvolvedores que vêem no crescimento exponencial do número de dispositivos móveis vendidos pelo mundo, uma oportunidade de utilizar a mobilidade para aprimorar a experiência do usuário e trazer conforto na utilização da plataforma móvel.

Ver detalhes »

Administração de Redes Windows

A configuração de rede no Windows é um assunto bastante conhecido, pois a configuração é bastante similar entre as diferentes versões do sistema e a configuração gráfica torna tudo mais simples. Este curso abordará o emprego pratico de configurações de redes Windows.

Ver detalhes »

Serviços de Diretórios LDAP

Neste curso de OpenLDAP básico o aluno vivenciará em cenários reais, como implementar de forma totalmente prática um servidor de diretórios integrando os principais serviços usados em empresas. As integrações Samba com LDAP, além de Postfix, Squid Proxy, FTP e Apache são a base dos laboratórios práticos.

Ver detalhes »

Endereçamento de Redes IPv6

O protocolo IPv6 apresenta como principal característica e justificativa maior para o seu desenvolvimento, o aumento no espaço para endereçamento. Por isso, é importante conhecermos as diferenças entre os endereços IPv4 e IPv6, saber reconhecer a sintaxe dos endereços IPv6 e conhecer os tipos de endereços IPv6 existentes e suas principais características.

Ver detalhes »

Segurança Computacional

Os crimes eletrônicos se tornaram uma prática comum, afetando usuários de sistemas computacionais e empresas de pequeno, médio e grande porte. Este curso permite identificar os elementos envolvidos em ataques assim como ensinará ao aluno como se proteger.
